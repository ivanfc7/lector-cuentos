# Escritos Literarios: Blog literario

Caracteristicas:

- ✅ Estilo minimalista
- ✅ SEO-amigable con URLs canonicas
- ✅ Soporte Markdown & MDX
- ✅ Realizado en Astro 

## Estructura del proyecto

El contenido de los cuentos y poemas junto a sus imagenes desde las carpetas:
- assets/ilustraciones-escritos
- content/cuento
- content/poema

```text
├── public/
├── src/
│   ├── assets/
│   ├── components/
│   ├── content/
│   ├── layouts/
│   └── pages/
|   └── styles/
├── astro.config.mjs
├── README.md
├── package.json
└── tsconfig.json
```
